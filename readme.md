We study the Equations of State for hardcore bosons in this project.

The hardcore bosons free model, like many others, is exact solvable, so one already has the partition function, for this model.
So the EoS should be derived from the partition functions cleanly.

What we are interested here is how experimentally one may verify this.

The plan is to:
(1) first investigate what perturbations we add to the model can thermalize the model, as integrable models don't thermalize.
(2) then, we time evolve the perturbed model for a long time to get the thermal state.
(3) once we have thermal state, we can try to measure EoS by doing linear response theory or some high temperature expansion.

The target Hamitlonian we work with is
$$H = -t \sum_i b^\dagger_i b_{i+1} + h.c. $$
